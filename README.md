# Zeronote

Fully encrypted notepad with rich text support. No logins, no tracking, ever.

## Features

- Encrypted Notes
- Hashed Note Names
- Rich Text Support
- No Account Required
- Zero Tracking 
- Available online or self host
- Simple, clean interface
- Responsive design
